# Arrancar la aplicacion

Esta aplicacion es un servidor HTTP sencillo escrito en Python. No necesita instalar dependencias externas.

## Ejecutar

Desde este directorio:

```bash
python3 app.py
```

Cuando arranque, veras:

```text
Servidor iniciado en http://127.0.0.1:8000
```

Abre esta URL en el navegador:

```text
http://127.0.0.1:8000
```

La respuesta esperada es:

```text
hola mundo!
```

## Detener

Pulsa `Ctrl+C` en la terminal donde esta corriendo el servidor.
